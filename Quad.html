Welcome, to the Quad Game!

This is a survival game.

The game works like a normal text game, there are choices and buttons for you to press. It all depends on your current situation. For now, you need to type your name so you can start the adventure.

Created by: Illidan Lucis Lecter S. Tria

And now, for the description of the game: This game is about surviving the "Quad". The "Quad" is a shadow that lurks into your house and follow "you". Lets see if you can survive the "Quad". Enjoy! :)
